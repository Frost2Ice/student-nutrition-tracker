# ระบบติดตามภาวะโภชนาการนักเรียน · Student Nutrition Tracker

แอปออฟไลน์สำหรับ **ครูอนามัยโรงเรียน** ใช้บันทึกน้ำหนัก/ส่วนสูง ประเมินภาวะโภชนาการ
ตามเกณฑ์การเจริญเติบโต **กรมอนามัย พ.ศ. 2564** (อายุ 2–18 ปี) และติดตามการเติบโตของนักเรียน
ตลอดปีการศึกษา — ทำงานได้เต็มรูปแบบโดยไม่ต้องต่ออินเทอร์เน็ต

> An offline-first web app for Thai school health teachers: record weight/height,
> classify nutritional status against the Thai Department of Health growth reference
> (B.E. 2564, ages 2–18), track growth over time, and handle the yearly academic cycle.
> Runs fully offline as a single self-contained HTML file.

---

## ✨ คุณสมบัติหลัก · Features

- **บันทึกการวัดแบบตาราง** — เลือกห้อง เลือกครั้งที่วัด แล้วกรอก/แก้ไขน้ำหนัก-ส่วนสูงได้ในตารางเดียว
  เห็นผลการประเมิน **4 เกณฑ์แยกคอลัมน์** ทันที (น้ำหนัก/อายุ · ส่วนสูง/อายุ · น้ำหนัก/ส่วนสูง · สูงดีสมส่วน)
- **เก็บผลล่าสุดต่อรอบ** — หนึ่งผลการวัดต่อ (นักเรียน · ปี · ภาคเรียน · ครั้งที่) บันทึกซ้ำ = อัปเดตทับ ไม่เพิ่มซ้ำ
- **โปรไฟล์รายคน** — ข้อมูลส่วนตัว ประวัติการวัดทั้งหมด กราฟการเติบโต 3 แบบ (WFH/HFA/WFA)
  พร้อมเส้นเกณฑ์อ้างอิง และส่งออกเป็น PDF
- **นำเข้า/ส่งออก Excel** — ขั้นตอนเดียวกันทั้งแอป: ดาวน์โหลดแม่แบบ → เลือกไฟล์ → ตรวจสอบตัวอย่าง
  (เพิ่มใหม่ / อัปเดต / มีปัญหา) → ยืนยัน รองรับไฟล์จริงที่ Excel แปลงรูปแบบวันที่
- **เลื่อนชั้นปีการศึกษา** — ตัวช่วยแบบทีละขั้น สำรองข้อมูลก่อน ส่งออกผู้จบการศึกษาพร้อมประวัติ ก่อนนำออกจากทะเบียน
- **โครงสร้างชั้นเรียนยืดหยุ่น** — กำหนดจำนวนห้องต่อชั้น ชั้นที่ 0 ห้องถือว่าไม่มี ป้องกันการลบห้องที่ยังมีนักเรียน
- **สำรอง/กู้คืนข้อมูล** — ไฟล์สำรองเป็น JSON เปิดกลับมาใช้เมื่อย้ายเครื่องได้

## 🔒 ความเป็นส่วนตัว · Privacy

ข้อมูลทั้งหมดเก็บใน `localStorage` ของเครื่องนี้เท่านั้น ไม่มีการส่งข้อมูลออกนอกเครื่อง
ตัวแอปที่ build แล้วเป็นไฟล์ HTML ไฟล์เดียว เปิดได้บน PC / แท็บเล็ต / iPad โดยไม่ต้องติดตั้งหรือต่อเน็ต

## 🧰 เทคโนโลยี · Tech Stack

Vue 3 (`<script setup>` + TypeScript) · Pinia · Vite · Chart.js · SheetJS (xlsx) ·
Vitest · บันเดิลเป็นไฟล์เดียวด้วย `vite-plugin-singlefile`

## 🚀 เริ่มต้นใช้งาน · Getting Started

ต้องมี Node.js 18+ (พัฒนาด้วย Node 22)

```bash
npm install        # ติดตั้ง dependencies
npm run dev        # เปิดเซิร์ฟเวอร์พัฒนา → http://localhost:5173
npm run build      # ตรวจชนิดข้อมูล + build เป็น dist/index.html ไฟล์เดียว
npm run preview    # ดูตัวอย่างไฟล์ที่ build แล้ว
npm test           # รันชุดทดสอบ (Vitest)
```

ผลลัพธ์จาก `npm run build` คือ `dist/index.html` ที่รวมทุกอย่างไว้ในไฟล์เดียว
คัดลอกไปเปิดบนเครื่องเป้าหมายได้ทันที (ดับเบิลคลิกเปิดในเบราว์เซอร์)

## 📂 โครงสร้างโปรเจกต์ · Project Structure

```
src/
  domain/        ตรรกะหลัก ไม่ผูกกับ framework + มี unit test
    nutrition/   เครื่องมือประเมินภาวะโภชนาการ (แหล่งความจริงเดียว — engine.ts)
    date/        วันที่แบบพุทธศักราช (ว/ด/ปป พ.ศ.)
    transfer/    นำเข้า/ส่งออก Excel (xlsx) และสำรองข้อมูล (JSON)
    grade/       ลำดับชั้นเรียนและการเลื่อนชั้น
    validation/  กฎตรวจสอบข้อมูล
  features/      หน้าจอแต่ละส่วน (บันทึกการวัด นักเรียน รายงาน ตั้งค่า ฯลฯ)
  components/    ส่วนประกอบที่ใช้ซ้ำ (Dialog, ImportDialog, Stepper)
  stores/        Pinia store (data.ts)
tests/           ชุดทดสอบ domain + store
mockdata/        ข้อมูลตัวอย่างทั้งโรงเรียน + ไฟล์ Excel สำหรับทดสอบการนำเข้า
docs/            สเปก แผนงาน และเอกสารผลิตภัณฑ์/ดีไซน์
```

## 🧪 ข้อมูลตัวอย่าง · Sample Data

`mockdata/` มีข้อมูลทั้งโรงเรียน (สร้างแบบ deterministic) สำหรับลองใช้งานทุกฟีเจอร์

```bash
node mockdata/generate.mjs
```

- `mockdata/sample-school.json` — เปิดผ่าน **ตั้งค่า → เปิดข้อมูลสำรอง**
- `mockdata/import-samples/` — ไฟล์ roster + measure (.xlsx) สำหรับทดสอบขั้นตอนนำเข้าจากเครื่องเปล่า
  (นำเข้ารายชื่อก่อน แล้วจึงนำเข้าผลการวัด)

## 📐 หลักการออกแบบ · Design Notes

- **แหล่งความจริงเดียว** — การประเมินทั้งหมดผ่าน `src/domain/nutrition/engine.ts`
  โซนสีบนกราฟต้องตรงกับผลการจัดกลุ่มเสมอ (มี invariant test คุม)
- **เกณฑ์ทางการ 4 ด้านเสมอ** — น้ำหนัก/อายุ · ส่วนสูง/อายุ · น้ำหนัก/ส่วนสูง · สูงดีสมส่วน แสดงแยกกัน ไม่รวบ
- **ภาษาไทยเป็นหลัก** ออกแบบให้ครูที่ไม่ถนัดเทคโนโลยีใช้ได้อย่างมั่นใจ ลดความกังวลเรื่องข้อมูลหาย
- **ปฏิทินไทย** — วันที่แบบ พ.ศ. และปีการศึกษา ≠ ปีปฏิทิน (ภาคเรียน 2 คาบเกี่ยว ม.ค.–มี.ค. ของปีถัดไป)
